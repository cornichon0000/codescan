# Streamlit Project

이 프로젝트는 Streamlit을 사용한 웹 애플리케이션입니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 실행 방법

다음 명령어로 애플리케이션을 실행할 수 있습니다:
```bash
streamlit run app.py
```

## 기능

- 기본적인 데이터 시각화
- 사용자 입력 처리
- 반응형 웹 인터페이스

## 프로젝트 구조

- `app.py`: 메인 애플리케이션 파일
- `requirements.txt`: 필요한 패키지 목록
